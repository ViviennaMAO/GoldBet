
You need to add the following secrets to your GitHub Repository:

1. **GOLD_API_KEY**: Your GoldAPI.io API Key.
2. **SUPABASE_SERVICE_ROLE_KEY**: Your Supabase Service Role Key.
3. **PROXY_URL** (Optional but recommended): A valid proxy URL (e.g., `http://user:pass@host:port`) if you want to bypass potential IP blocking.
